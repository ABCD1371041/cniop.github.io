# diyp直播源支持分组和密码设置

分组格式如：央视频道,#genre#

密码格式如：央视频道_“密码”,#genre# （海外_8888,#genre#）

每个电视频道只能在一行，频道名称和直播源之间使用半角逗号（，）分隔，多个源之间使用“#”分隔

可以为每个源指定名称如：源地址+$+源名称
